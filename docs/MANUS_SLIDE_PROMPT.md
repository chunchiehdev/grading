    ## PRESENTATION DESIGN SPECIFICATIONS

    ### Color Scheme
    - **Primary Blue**: #2563EB (Trust, intelligence)
    - **Primary Purple**: #9333EA (Innovation, AI)
    - **Success Green**: #10B981 (Achievement)
    - **Alert Orange**: #F97316 (Attention, improvement)
    - **Light Gray**: #F3F4F6 (Background)
    - **Dark Gray**: #1F2937 (Text)
    - **White**: #FFFFFF (Cards, contrast)

    ### Typography
    - **Headings**: Bold, sans-serif (Montserrat, Inter, or similar)
    - **Body**: Clean sans-serif (Open Sans, Roboto)
    - **Size Hierarchy**:
    - Title slides: 60-80pt
    - Main headings: 44-54pt
    - Subheadings: 28-36pt
    - Body text: 18-24pt
    - Small text: 14-16pt

    ### Animations (Manus Standard)
    - **Entrance**: Fade, Slide, Scale (0.3-0.5 seconds)
    - **Emphasis**: Pulse, Highlight, Grow
    - **Exit**: Fade (smooth transitions between slides)
    - **Timing**: Stagger elements for visual interest
    - **Pace**: Not too fast (allow reading time)

    ### Fonts to Use
    - Titles: **Montserrat Bold** or **Inter Bold**
    - Subtitles: **Open Sans Semibold**
    - Body: **Open Sans Regular**
    - Code/Data: **Roboto Mono**

    ### Icon Style
    - Use consistent icon set (Font Awesome, Material Icons, or Feather)
    - Size: 24-48pt for slide content
    - Color: Match theme colors
    - Keep minimal and clean
